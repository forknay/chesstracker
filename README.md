# chesstracker

Bloomberg-terminal-style dashboard tracking chess.com progress for a group of friends.

## Usage

Open `index.html` in any browser. No build step, no server, no dependencies to install.

Data refreshes automatically every 2 minutes (profiles/stats) and 10 minutes (game archives). Archive data is cached in localStorage.
